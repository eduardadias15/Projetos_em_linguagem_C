# Projetos_em_linguagem_C
Reporitório criado para postar codigos na liguagem C que estão sendo praticados e desenvolvidos por mim na faculdade de ciência da computação.

📚 Sobre
Este repositório reúne exercícios, desafios e pequenos projetos feitos nas aulas de programação da faculdade. O objetivo é registrar meu progresso e manter um histórico do meu aprendizado.

🗂️ Organização
Os arquivos estão organizados por assunto ou lista de exercícios, conforme as atividades propostas em aula.

🛠️ Tecnologias

Linguagem: C
Compilador: GCC

📌 Observações
Repositório em constante atualização conforme avanço nas disciplinas do curso.
